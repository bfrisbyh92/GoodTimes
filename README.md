# GoodTimes

## Mern Stack Social Media App to share good moments in life

### Tech Used in this build

- MongoDB
- Jsonwebtokens
- Bcryptjs
- Pagination
- Express.js
- Google Login Authentication
- Traditional email/hashed password authentication
- Material-UI for CSS
- SVG background image
